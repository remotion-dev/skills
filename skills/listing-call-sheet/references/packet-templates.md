# Content Plan + Three-Packet Templates

These are the output skeletons — but **the output itself is never a skeleton.** Fill every field with real, specific content. The only legitimate blanks are (a) clip IDs you assign, (b) genuinely unknowable post-sale data in the case-study shell, and (c) IDs/URLs the user must supply (GHL number, QR target) — mark those `[…TBD]`. Everything else is written out in full. A packet that reads like a form template has failed; it should read like a working document a real person picks up and acts on.

Keep clip IDs identical across all three packets — they are the join key.

Plain-language rule (applies to every packet): write directions a non-jargon reader can act on. Lead with plain English, abbreviation in parentheses. "Medium shot (waist-up), camera slowly walks toward the door on a gimbal." Never "MS gimbal push." The shot glossary (`shot-glossary.md`) goes at the top of the videographer packet.

## PART 1 — LISTING CONTENT PLAN

The content plan is the *strategy layer* — what we're making from this one shoot and why. Each item below has a one-line label so Graeham knows exactly what he's reading.

```
# LISTING CONTENT PLAN — <Address>
Price: $<x>  |  <beds>bd/<baths>ba  |  <sqft> sqft  |  <neighborhood>
Go-live: <date>   Shoot: <date>   Orientation: <portrait | landscape | both>
Avatar look: <chosen look>   Voice: ElevenLabs clone (default)
```

### A. The listing video

Write it as TWO labeled lines, each self-explanatory:

```
LONG-FORM listing video  (<16:9 landscape | 9:16 portrait>)
• THE ARC (creative summary — NOT the words Graeham says): a 2–3 sentence description of how
  the finished video flows start to finish, so everyone sees the shape. The actual word-for-word
  script lives in the AGENT packet. Example: "Opens on Graeham at the curb, moves inside to the
  kitchen hero moment, out to the ADU, then the yard and solar, and closes on walkability with a
  text-keyword call to action." This line is the storyboard in a sentence, not narration.

SHORT-FORM cut  (<9:16 portrait>)
• This is a shorter edit built ENTIRELY from clips already filmed for the long-form video — no
  extra shooting. List the exact clips it reuses BY THEIR CLIP ID, and say what each is, e.g.:
  "Reuses already-filmed clips: COOLEY-A01 (the opener), COOLEY-L02 (waterfall-island reveal),
  COOLEY-G01 (price card), COOLEY-A04 (the call-to-action)." Then one line on the structure.
```

### B. PropReach derivatives (drafted as BRIEFS — handed off to PropReach, not produced here)

Each is a short brief, fully written (angle, audience, hook line, which already-filmed clip IDs it reuses, the GHL keyword CTA). Four of them:

```
1. SELLER-FOCUSED AD — angle, target audience, hook line (write the actual hook), reused clip IDs, GHL keyword.
2. BRAND-AWARENESS AD — angle (Graeham / EPA expertise, NOT this listing), actual hook line, reused clip IDs, GHL keyword.
3. POSTCARD CONCEPT — front image idea, the actual headline, [QR PLACEHOLDER → retargeting URL TBD], reverse-side copy written out.
4. CASE-STUDY SHELL — skeleton to fill AFTER the property sells. Every post-sale field stays a placeholder:
   [DOM TBD] · [SALE PRICE TBD] · [OVER/UNDER LIST % TBD] · [# OFFERS TBD] · [buyer-story angle TBD].
   State clearly: do not publish until close of escrow.
```

## PART 2 — VIDEOGRAPHER PACKET (numbered shot list)

Open with the shot glossary (`shot-glossary.md`), then the three categories. Every field-capture shot carries **seven** things now (composition is the addition the field crew asked for):

1. **Clip ID** · 2. **What's in frame** (specific) · 3. **Composition** (how much of the subject, camera height, sky-vs-ground ratio, exactly where to stand) · 4. **Camera move** (plain English) · 5. **Shot size** (plain English) · 6. **Location** · 7. **Hold** (seconds).

```
# VIDEOGRAPHER PACKET — <Address>   Shoot: <date>
For: <videographer name/email>
Orientation: <…>   Export each clip as <CLIPID>_<orientation>.<ext>. Nothing else in the filename.

### SHOT GLOSSARY  (paste shot-glossary.md here so directions are readable cold)

NOTE: Category (c) avatar-source shots are STRICTER than B-roll — locked tripod, even light, held
framing, 2+ minutes (see the callout). The rest can be filmed loose.
```
