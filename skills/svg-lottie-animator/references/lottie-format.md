## 顶层结构

```json
{
  "v": "5.9.0",        // Lottie 版本
  "fr": 25,            // 帧率 (frames per second)
  "ip": 0,             // 入点帧 (in-point)
  "op": 90,            // 出点帧 (out-point)，总帧数 = op - ip
  "w": 750,            // 画布宽度 (像素)
  "h": 720,            // 画布高度 (像素)
  "nm": "动画名称",     // 合成名称
  "ddd": 0,            // 是否 3D (0=2D, 1=3D)
  "assets": [],        // 资源数组（图片、预合成等）
  "layers": [],        // 图层数组
  "markers": []        // 标记数组（可选）
}
```

**时长计算**: `duration_seconds = (op - ip) / fr`

## 资源 (Assets)

### 图片资源

```json
{
  "id": "image_0",              // 唯一 ID
  "w": 128,                     // 宽度
  "h": 128,                     // 高度
  "u": "",                      // 路径前缀（内嵌时为空）
  "p": "data:image/png;base64,...",  // 图片数据或文件名
  "e": 1                        // 是否内嵌 (1=内嵌base64, 0=外部文件)
}
```

### 预合成资源 (Precomposition)

```json
{
  "id": "comp_0",       // 唯一 ID，被图层的 refId 引用
  "nm": "合成名称",
  "fr": 25,             // 帧率
  "layers": [...]       // 此合成内的图层数组
}
```

## 图层 (Layers)

### 图层通用属性

```json
{
  "ddd": 0,         // 3D 图层 (0=2D)
  "ind": 1,         // 图层索引 (唯一标识)
  "ty": 4,          // 图层类型
  "nm": "图层名称",
  "sr": 1,          // 时间伸缩因子
  "ks": {},         // 变换属性 (Transform)
  "ao": 0,          // 自动旋转 (auto-orient)
  "ip": 0,          // 入点
  "op": 90,         // 出点
  "st": 0,          // 开始时间偏移
  "bm": 0           // 混合模式 (0=normal)
}
```

### 图层类型 (ty)

| ty | 类型 | 特有属性 |
|----|------|---------|
| 0 | 预合成 | `refId`, `w`, `h` |
| 1 | 纯色 | `sw`, `sh`, `sc` (宽/高/颜色) |
| 2 | 图片 | `refId` (引用 assets 中的图片) |
| 3 | 空图层 | 常用作父级控制 |
| 4 | 形状 | `shapes` 数组 |
| 5 | 文字 | `t` (文字数据) |

### 遮罩相关属性

```json
{
  "td": 1,    // 定义为遮罩源 (Track Matte Definition)
  "tt": 1     // 使用上方图层作为遮罩 (Track Matte Type: 1=Alpha, 2=Alpha反转, 3=亮度)
}
```

### 预合成图层示例

```json
{
  "ty": 0,
  "refId": "comp_0",    // 引用 assets 中的合成 ID
  "w": 128,              // 合成尺寸
  "h": 128,
  "ks": {
    "p": {"a": 0, "k": [375, 360, 0]},   // 在父级中的位置
    "a": {"a": 0, "k": [64, 64, 0]},     // 锚点（通常为合成中心）
    "s": {"a": 0, "k": [100, 100, 100]}   // 缩放
  }
}
```

## 变换属性 (Transform)

每个图层的 `ks` 对象包含以下变换属性：

```json
"ks": {
  "o":  {"a": 0, "k": 100, "ix": 11},           // 透明度 Opacity (0-100)
  "r":  {"a": 0, "k": 0, "ix": 10},             // 旋转 Rotation (度)
  "p":  {"a": 0, "k": [375, 360, 0], "ix": 2},  // 位置 Position [x, y, z]
  "a":  {"a": 0, "k": [0, 0, 0], "ix": 1},      // 锚点 Anchor Point [x, y, z]
  "s":  {"a": 0, "k": [100, 100, 100], "ix": 6}  // 缩放 Scale [x%, y%, z%]
}
```

- `a`: 是否有动画 (0=静态, 1=有关键帧)
- `k`: 静态值或关键帧数组
- `ix`: 属性索引（内部标识）

## 关键帧 (Keyframes)

### 单值属性关键帧（旋转、透明度等）

```json
"r": {
  "a": 1,
  "k": [
    {
      "i": {"x": [0.833], "y": [0.833]},   // 入缓动（贝塞尔控制点）
      "o": {"x": [0.167], "y": [0.167]},   // 出缓动
      "t": 0,                               // 关键帧时间（帧数）
      "s": [0]                              // 起始值
    },
    {
      "i": {"x": [0.833], "y": [0.833]},
      "o": {"x": [0.167], "y": [0.167]},
      "t": 30,
      "s": [360]
    },
    {
      "t": 60,                              // 最后一个关键帧只需 t 和 s
      "s": [0]
    }
  ]
}
```

### 多值属性关键帧（位置等）

```json
"p": {
  "a": 1,
  "k": [
    {
      "i": {"x": 0.833, "y": 0.833},
      "o": {"x": 0.167, "y": 0.167},
      "t": 0,
      "s": [100, 200, 0],                  // 起始位置 [x, y, z]
      "to": [50, 0, 0],                    // 出切线 (运动路径贝塞尔)
      "ti": [-50, 0, 0]                    // 入切线
    },
    {
      "t": 30,
      "s": [400, 200, 0]
    }
  ]
}
```

**运动路径**: `to`/`ti` 控制位置动画的贝塞尔曲线路径，使运动轨迹变为曲线而非直线。

### 路径关键帧（形状变形）

```json
"ks": {
  "a": 1,
  "k": [
    {
      "i": {"x": 0.833, "y": 0.833},
      "o": {"x": 0.167, "y": 0.167},
      "t": 0,
      "s": [{
        "i": [[0,0], [-10, 5]],    // 入控制点
        "o": [[5, -10], [10, 0]],   // 出控制点
        "v": [[100, 200], [300, 100]], // 顶点坐标
        "c": false                   // 是否闭合路径
      }]
    },
    {
      "t": 30,
      "s": [{
        "i": [[0,0], [-5, 10]],
        "o": [[10, -5], [5, 0]],
        "v": [[150, 150], [350, 150]],
        "c": false
      }]
    }
  ]
}
```

### 缓动曲线参考

贝塞尔缓动由 `i`(in) 和 `o`(out) 控制点定义：

| 效果 | o (出) | i (入) | 描述 |
|------|--------|--------|------|
| 线性 | `{x:0, y:0}` | `{x:1, y:1}` | 匀速运动 |
| 缓入缓出 | `{x:0.167, y:0.167}` | `{x:0.833, y:0.833}` | 平滑启停，最常用 |
| 快速缓入 | `{x:0.333, y:0}` | `{x:0.833, y:0.833}` | 快速启动 |
| 快速缓出 | `{x:0.167, y:0.167}` | `{x:0.333, y:1}` | 快速停止 |
| 弹跳 | 需要多个关键帧模拟，振幅递减 | | |

### 保持帧 (Hold Keyframes)

```json
{
  "t": 10,
  "s": [100],
  "h": 1          // h=1 表示保持到下一关键帧，无过渡
}
```

## 形状 (Shapes)

形状图层 (ty=4) 的 `shapes` 数组包含形状组：

### 形状组 (Group)

```json
{
  "ty": "gr",           // 组
  "it": [...],          // 组内元素
  "nm": "组名称",
  "np": 2,              // 属性数量
  "bm": 0               // 混合模式
}
```

### 常见形状类型

| ty | 类型 | 说明 |
|----|------|------|
| `gr` | 组 (Group) | 包含子形状和变换 |
| `sh` | 路径 (Shape) | 贝塞尔路径 |
| `rc` | 矩形 (Rect) | 矩形形状 |
| `el` | 椭圆 (Ellipse) | 椭圆形状 |
| `fl` | 填充 (Fill) | 颜色填充 |
| `st` | 描边 (Stroke) | 描边样式 |
| `tr` | 变换 (Transform) | 组的变换属性 |
| `rd` | 圆角 (Round Corners) | 圆角修饰 |
| `mm` | 合并路径 (Merge) | 路径布尔运算 |

### 填充示例

```json
{
  "ty": "fl",
  "c": {"a": 0, "k": [0.5, 0.5, 0.8, 1], "ix": 4},  // RGBA (0-1)
  "o": {"a": 0, "k": 100, "ix": 5},                    // 透明度
  "r": 1,                                               // 填充规则
  "bm": 0,
  "nm": "填充 1"
}
```

### 描边示例

```json
{
  "ty": "st",
  "c": {"a": 0, "k": [0, 0, 0, 1], "ix": 3},    // 颜色
  "o": {"a": 0, "k": 100, "ix": 4},               // 透明度
  "w": {"a": 0, "k": 4.667, "ix": 5},             // 线宽
  "lc": 2,                                         // 线端 (1=butt, 2=round, 3=square)
  "lj": 1,                                         // 线连接 (1=miter, 2=round, 3=bevel)
  "ml": 4                                          // miter limit
}
```

### 组变换

每个形状组最后通常有一个 `tr` 变换：

```json
{
  "ty": "tr",
  "p": {"a": 0, "k": [0, 0], "ix": 2},     // 位置
  "a": {"a": 0, "k": [0, 0], "ix": 1},     // 锚点
  "s": {"a": 0, "k": [200, 200], "ix": 3}, // 缩放 (注意：200 = 2x)
  "r": {"a": 0, "k": 0, "ix": 6},          // 旋转
  "o": {"a": 0, "k": 100, "ix": 7}         // 透明度
}
```

## 遮罩 (Masks)

### Alpha 遮罩

遮罩通过相邻图层的 `td` 和 `tt` 属性实现：

```json
// 图层顺序：遮罩定义层必须在被遮罩层之前
{
  "ind": 1,
  "ty": 4,
  "nm": "遮罩形状",
  "td": 1,           // 此图层定义遮罩
  "shapes": [...]    // 遮罩形状（白色区域可见）
},
{
  "ind": 2,
  "ty": 4,
  "nm": "被遮罩内容",
  "tt": 1,           // 使用上方图层作为 Alpha 遮罩
  "shapes": [...]
}
```

### tt 遮罩类型

| tt 值 | 类型 | 说明 |
|-------|------|------|
| 1 | Alpha | 遮罩白色区域显示 |
| 2 | Alpha 反转 | 遮罩白色区域隐藏 |
| 3 | 亮度 | 基于亮度遮罩 |
| 4 | 亮度反转 | 基于亮度反转遮罩 |
