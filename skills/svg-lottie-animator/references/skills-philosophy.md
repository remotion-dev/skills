# 动画哲学与运动隐喻

## 动画哲学 (核心灵魂)

在编写任何动画代码前，必须先定义动画的“性格”：

1. **品牌个性**: 
   - **专业**: 精准、简洁、低调
   - **活泼**: 动感、色彩鲜艳、充满弹性
   - **优雅**: 丝稳、慢节奏、高阶质感
   - **有力**: 快速、明确、极具冲击力

2. **情感响应**:
   - **信任**: 稳定的节奏，无突兀跳跃
   - **兴奋**: 快速入场，配合强力回弹
   - **平静**: 类似呼吸的慢循环
   - **紧迫**: 急促的重复与线性切入

3. **运动隐喻 (拟物化逻辑)**:
   - **像水一样**: 流畅的入场/退场，无缝衔接
   - **像弹簧一样**: 极具弹性，包含过冲与阻尼
   - **像空气一样**: 轻盈无感，主要用于环境氛围

## 职业动画师视角

### 什么时候使用？

| 隐喻 | 场景案例 | 核心缓动建议 |
|------|---------|-------------|
| **弹力球** | 成功提示、金币弹出 | Bounce (`[0.34, 1.56]`, `[0.64, 1]`) |
| **深呼吸** | 待机状态、加载中 | Cubic In-Out (`[0.645, 0.045]`, `[0.355, 1]`) |
| **快速抓取** | 搜索完成、列表展开 | Expo Out (`[0.19, 1]`, `[0.22, 1]`) |
| **磁铁吸附** | 选项吸附、开关切换 | Snappy (`[0.2, 0]`, `[0, 1]`) |

### 缓动家族选择指南

根据物理特性和视觉冲击力选择合适的缓动家族：

1. **Sine / Quad**: 适合**细微、优雅**的动作（如：呼吸感、柔软的漂浮）。
2. **Cubic / Quart**: 适合**标准 UI 交互**，具有明显的动感变化。
3. **Quint / Expo**: 适合**极速冲击**（如：闪现进入、瞬间弹出），视觉对比极强。
4. **Circ**: 适合**非对称**节奏，具有明显的“突然启动”或“极速收尾”特征。
5. **Back (过冲)**: 赋予动画“拟人化”和“弹性”，适合**活泼、友好**的品牌性格。
6. **Bounce / Elastic**: 适合需要**物理模拟**的场景（如：真实下落、果冻效果）。

### 专业技巧
- **间距感 (Spacing)**: 不是简单的速度，而是每一帧之间位移的变化率。
- **弧线运动**: 避免直线位移，使用 `to`/`ti` 创造自然的抛物线。
- **重力模拟**: 下落加速，上升减速。
