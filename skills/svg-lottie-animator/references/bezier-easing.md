# 专业贝塞尔缓动曲线库

## 基础概念

贝塞尔曲线控制关键帧之间的插值方式：
- **X 轴**: 时间 (0=当前帧, 1=下一帧)
- **Y 轴**: 值 (0=当前值, 1=目标值)
- `o` (out): 从当前帧出发的控制点
- `i` (in): 进入下一帧的控制点

```json
{"t":0, "s":[0], "o":{"x":[0.33],"y":[0]}, "i":{"x":[0.67],"y":[1]}}
```

## 极值效果

- **过冲 (y>1)**: 值超过目标后回弹 → `"i":{"x":[0.5],"y":[1.2]}`
- **预备 (y<0)**: 先反向再正向 → `"o":{"x":[0.5],"y":[-0.1]}`
- **保持帧**: `"h":1` 冻结到下一帧，无过渡

## 使用原则

1. **一致性**: 同类动画使用相同缓动
2. **入场用 Ease Out**, 退场用 Ease In, 循环用 Ease In-Out
3. **过冲控制在 20-30%** 以内保持专业感
4. **弹跳/弹簧** 用于活泼/游戏风格
5. **Material/Snappy** 用于 UI 交互
