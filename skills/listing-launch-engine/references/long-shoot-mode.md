# Long-Shoot Mode — maximize a 4–5 hour videographer window

Wesley usually arrives **3+ hours before Graeham** and the on-camera block with Graeham runs about **2 hours**, so a single trip is often a **4–5 hour window**. That is enough to capture a month-plus of content, not one video. When the shoot window is long, switch from the minimum shot list to this plan.

## Step-0 add: ask the shoot window

Before building, ask: **"How long is Wesley on site, and how many of those hours is he solo before/after you?"** That sets the mode:

- **Short (~the Graeham window only, 1–2 hrs):** the lean shot list — the listing video + the 4-photo status set + a handful of B-roll. Cap forward/bank shots at 5.
- **Long (3–5 hrs, Wesley solo for part):** this mode. Time-blocked plan below, 35–50 clips, and the **forward/bank-shot cap lifts** — capture as much demand-backed and evergreen B-roll as the window allows (every clip still tied to a real planned video or a content pillar; never random spares).

## The time-blocked plan

Split the shoot into a **Wesley-solo block** and a **Graeham block**, so nobody waits on anybody.

### Block A — Wesley solo (the first 3 hours, no agent needed)
| Pass | What to get | Target clips | Notes |
|---|---|---|---|
| **Drone / aerial** | House + lot from above, roofline, the block, approach down the street, the neighborhood context (parks, bay, trails) | 5–8 | **Airspace check first** — East Palo Alto sits under Palo Alto Airport (KPAO) controlled airspace. Wesley must clear LAANC / check B4UFLY before flying. If no authorization, skip drone and get elevated/high-angle ground shots instead. |
| **Neighborhood B-roll sweep** | The lifestyle around the home: bay trail, Cooley Landing, University Ave retail, parks, transit, coffee, streetscapes, signage | 8–12 | Feeds neighborhood videos + the launch's Lifestyle phase + the evergreen bank. **Fair Housing: places, amenities, commute — never who lives there.** |
| **Exteriors (golden hour)** | Front elevation (the 4-photo set), rear, sides, the lot/yard, the garage exterior, entry/curb | 6–8 | Plan the front-elevation pass for golden hour, sun behind camera. |
| **Interiors** | Every room, the flow, hero spaces (kitchen/living/primary), the refreshed features, ceiling/light | 10–15 | Full walkthrough coverage so the YouTube tour + POV + carousels are all covered. |
| **Detail / texture** | Fixtures, finishes, hardware, the concrete patio, the AC unit, anything that sells craft | 6–10 | Cutaways for every reel + the carousel. |
| **Twilight (if staying)** | Exterior dusk with the lights on | 2–4 | The premium hero look; only if the schedule allows. |

### Block B — Graeham on camera (the ~2 hours he's there)
| Pass | What to get | Target clips |
|---|---|---|
| **Talking-head segments** | The on-camera lines the launch needs that we are NOT rendering as an avatar (see `heygen-avatars.md` — most can be avatar-rendered, so keep this tight) | 3–6 |

| **Pieces needing him** | Sold-sign setup with the seller, any client moment | as needed |

## The B-roll bank (the long-shoot payoff)

With hours on site, bank a **neighborhood + evergreen library** that feeds the whole content calendar, not just this listing. Every bank clip is category `B` (see `clip-id-convention.md`) and **must name the planned video or content pillar it feeds** — still demand-backed, just no longer capped at 5 in long mode. Typical pillars it serves:

- Monthly EPA / Peninsula market-update videos (need fresh local B-roll every cycle)
- Neighborhood guide / "Living in East Palo Alto" relocation videos
- Green-screen backdrops for talking-head reels
- AB 1482 / first-time-buyer explainers (street-level Peninsula B-roll)
- The next listing's "sold near you" farm content

Log the bank as its own table in the videographer + editor packets, each clip tagged to its destination.

## Content yield (put this on the dashboard)

Long mode should state the return on the trip so Graeham sees it at a glance:

> **This one shoot → ~[N] videos + [M] ads + a [K]-clip neighborhood B-roll bank.**

A 4–5 hour EPA cottage shoot typically yields on the order of **8–12 videos** (long-form tour, 4–6 reels, POV, neighborhood walkthrough, market-backdrop pieces), **4–6 ads** (Just-Listed, Disclosure, Open House, Sold→Worth, plus retargeting cuts), and a **15–25 clip B-roll bank**. Scale to the actual property and window; never pad — only count assets the captured footage genuinely supports.

## How this plugs into the build

- The launch slate (`launch-content-playbook.md`) defines the assets; long-shoot mode makes sure the shot list captures footage for ALL of them plus the bank.
- Avatar talking-heads come from `heygen-avatars.md` (most are rendered, not filmed) — so Block B stays short and Block A does the heavy lifting.
- Show the time blocks + the content-yield line on the dashboard, and split the videographer packet into Block A / Block B.
