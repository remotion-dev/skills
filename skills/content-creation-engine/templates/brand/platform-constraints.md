# Platform Constraints

Format specs and constraints per platform. Read before writing any script so you're optimizing for the right delivery.

## YouTube (long-form)

- **Target length:** 4-8 minutes for BOFU, 8-15 for edutainment deep-dive
- **Aspect ratio:** 16:9 (1080p or 4K)
- **Pacing:** 140-160 WPM spoken (cap 150 as planning baseline)
- **Chapters:** 4-6 chapters for long videos, 2-3 for short
- **Thumbnail:** high-contrast face + text (max 4 words)
- **Description:** first 3 lines visible above fold — put the value prop + CTA there
- **Algorithmic priority:** first 30s watch time, click-through rate on thumbnail

## YouTube Shorts / IG Reels / TikTok

- **Target length:** 15-60 seconds (sweet spot: 30-45)
- **Aspect ratio:** 9:16 vertical
- **First 2 seconds:** PURE HOOK — no logo, no intro, no music fade-in. Face + scroll-stopping statement.
- **Captions:** always burned in (most viewers watch muted)
- **Pacing:** aggressive — cut every 1-2 seconds max
- **CTA:** single ask, always at the end ("Comment EPA")

### Platform-specific tweaks

**IG Reels:** caption up to 2,200 chars, first 125 visible without "more" click. Hashtags 5-10 max, niche > broad.

**TikTok:** captions shorter (150-200 chars), hashtags 3-5. Pacing 10-15% faster than IG Reels. Native-text-overlay typography preferred.

**YT Shorts:** can be 60s+ but algorithm favors 30s. Vertical + burned captions + aggressive first 3 seconds.

## Instagram Carousel

- **Slides:** 8-10 (algorithm favors 8+)
- **Aspect ratio:** 4:5 (1080×1350)
- **Slide 1:** cover with headline + curiosity gap
- **Final slide:** CTA + contact
- **Save-bait:** mid-carousel should have a "reference" slide worth saving (checklist, formula, comparison)

## Blog (SEO + AEO)

- **Word count:** 1,000-1,800 for BOFU, 2,000-3,500 for deep guides
- **H-hierarchy:** one H1 (matches title), 4-6 H2s, H3s under each
- **AEO optimization:** each H2 should answer a clear question; first paragraph under the H2 is the direct answer (50-80 words)
- **Internal links:** 3-5 per post to related content
- **Schema markup:** Article + FAQ + LocalBusiness

## Newsletter

- **Total length:** 800-1,500 words for weekly
- **Structure:** hero/lead (1 topic), market update (stats), community news (2-3 items), featured listings (2-3), home-value CTA
- **Subject line:** 6-10 words, front-load the most interesting word
- **Preview text:** 30-90 chars that complement (don't repeat) subject

## GoogleMyBusiness Post

- **Length:** 150-300 words
- **CTA button:** always enabled (Learn More, Call, Book)
- **Image:** bright, face-forward if possible
- **Frequency:** weekly minimum (GMB rewards consistency)
- **Keywords:** city name + "real estate" or "realtor" in first 50 words

## Facebook

- **Extended caption length:** 200-400 words
- **Links:** Facebook suppresses external links — put them in first comment, not post body
- **Video:** upload native, don't link YouTube (huge algorithmic penalty for YT links)

## LinkedIn

- **Post length:** 300-500 words (sweet spot), up to 1,300
- **First 3 lines:** visible before "see more" — put the hook there
- **Hashtags:** 3-5 (more reduces reach)
- **Posting time:** Tue-Thu 8-10am PT hits professional audience
- **Tone:** authority + professional, more data-heavy than IG

## Ad Copy (Paid)

- **FB/IG:** 3 variants (hook-led, benefit-led, social-proof-led), 25-50 words each for body
- **Google Ads:** 30-char headlines × 3 + 90-char descriptions × 2
- **Landing page:** match ad copy messaging, reduce friction (one CTA, no navigation)

## Cross-posting matrix (high-level)

- **YT Long** → always the anchor, publish first
- **YT Short** → adapted from YT Long's strongest 30s
- **IG Reel #1** → hook-led variant of YT Short, captions overhauled
- **IG Reel #2** → data-led variant, second A/B
- **TikTok** → IG Reel #1 re-edited with faster pacing
- **Blog** → full explanation, links to YT Long
- **GMB** → short version of blog headline + link
- **Facebook** → extended caption + image
- **LinkedIn** → professional rewrite
- **Ad variants** → 3 versions for paid amplification
- **Newsletter** → weekly digest referring back to YT Long
