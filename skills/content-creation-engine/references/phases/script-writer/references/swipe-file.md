# Swipe File — Content That Actually Landed

**This is the highest-leverage file in the humor system, and it is mostly empty until Graeham fills it.** The model imitates concrete examples far better than it follows abstract rules. Showing it 15-25 real pieces that genuinely landed will do more for "funny and not generic" than any instruction in `humor-and-punch-up.md`.

The script-writer skims this file before writing any creative/short-form piece, to anchor on real material instead of regressing to the generic average.

---

## 👉 PASTE HERE (Graeham)

Drop 15-25 examples of content you think is genuinely funny, sharp, or hit hard. Yours OR creators you admire. Reels, scripts, captions, hooks, tweets — anything. Loose annotation is fine; even just the link plus "this one worked" helps. The more you add, the sharper the output.

Use this template per entry (copy the block):

```
### [short name]
- Link / source:
- Format: (Reel / Short / caption / hook / YouTube intro / tweet)
- The line(s) that land:
- Why it works (one line, your gut read):
- Technique (optional — map to humor-and-punch-up.md if you can):
- Transferable move (what's the reusable pattern here):
```

Tips for picking good ones:
- Include 3-5 of YOUR OWN best performers, even non-funny ones — they teach the model *your* cadence.
- Include a few from creators OUTSIDE real estate whose timing you love. Range helps.
- A mix of "made me laugh" and "made me stop scrolling / saved it" is ideal — both are the goal.
- Don't overthink the annotation. "Idk it's just funny" is a usable data point.
