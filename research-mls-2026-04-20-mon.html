Scraper Skill Specification

skills/mls-matrix-scraper/SKILL.md

A Chrome-automation skill that logs in to MyMLS and drives the Matrix → Stats panel. Configurable via CLI args or JSON config. Outputs CSV or JSON.

Supported stats (from Matrix → Stats → Customize):

  • Sale Price: Average, Median
  • List-to-Sale Price Ratio
  • Days on Market (Average, Median)
  • Active Inventory Count
  • Sold Count
  • Group by: Month, Quarter, Year

Filters: City (EP, RC, PA, MP), Property Type (Residential), Building Type (Single Family Home, Condo, Townhouse), Time Frame (Past 3 Years or custom). Credentials stored via Chrome SSO — skill does NOT handle passwords.

Methodology

When real: the scraper skill logs in via Chrome, navigates to Search → Matrix → Stats → Customize, sets Time Frame to Past 3 Years, Statistic to Sale Price Avg/Median/Ratio/DOM as requested, clicks Generate, downloads the CSV, and caches to outputs/mls-stats-{city}-{timestamp}.csv. Weekly content-calendar runs read the latest CSV for each city. The Opportunity Score's Audience Intent and Timeliness criteria can reference current MLS data to validate topic relevance.
